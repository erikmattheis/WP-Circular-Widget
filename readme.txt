== Installation ==

1. Upload the entire `wp-gsn-shortcodes` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the form to enter the Chain ID supplied by GSN by visiting the `GSN Circular Shortcodes` link in the admin menu
4. To output a GSN login form, use the shortcode [gsn_login]
5. To output a GSN shopping list, use the shortcode [gsn_shopping_list]
6. To output a GSN circular, use the shortcode [gsn_circular]
